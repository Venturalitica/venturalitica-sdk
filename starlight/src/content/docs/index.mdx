**AI Assurance through Compliance-as-Code.**

Venturalítica transforms your Python code into **Legal Evidence**. It automatically maps your technical metrics, data audits, and execution logs to the **EU AI Act (Articles 9-15)** without leaving your local environment.

> 📄 **Companion to the IEEE Computer 2026 paper** *AI Assurance That Compiles: Executable OSCAL for the EU AI Act* — [arXiv preprint (forthcoming)](https://arxiv.org/abs/<ARXIV-ID>) · [IEEE Xplore (forthcoming)](https://doi.org/<DOI>) · [Citation](https://github.com/Venturalitica/venturalitica-sdk/blob/main/CITATION.cff).

<Aside type="tip" title="Runs fully offline">
Venturalítica is a local-first SDK: install from PyPI and audit your AI
systems against a local `assessment_plan.oscal.yaml` policy. No network
calls, no hosted account required. The hosted SaaS (real-time policy
sync, multi-user evidence sharing) is forthcoming.
</Aside>

```python
# Run a full AI Assurance audit on the built-in loan scenario
results = vl.quickstart('loan')
```


## Key Features

| Feature | Description |
| :--- | :--- |
| **`enforce()`** | Audit datasets and models against OSCAL policies with 35+ built-in metrics. |
| **`monitor()`** | Wrap training runs with automatic evidence collection (BOM, hardware, carbon). |
| **Glass Box Dashboard** | 4-phase regulatory workflow: Identity, Policy, Verify, Report. |
| **Policy as Code** | Define assurance rules in OSCAL `assessment-plan` format. |
| **Column Binding** | Decouple policies from schemas via synonym-based column resolution. |
| **Local Sovereignty** | Zero-cloud dependency. All enforcement runs locally. |
| **Annex IV** | Auto-draft technical documentation from local traces. |

## Architecture

```text
┌─────────────────────────────────────────────────────────────┐
│                     Your ML Pipeline                        │
│                                                             │
│   DataFrame ──► vl.enforce(policy=…) ──► ComplianceResult[] │
│                        │                                    │
│                  OSCAL Policy                               │
│               (assessment-plan)                             │
└───────────┬─────────────────────────────────┬───────────────┘
            │                                 │
     ┌──────▼──────┐                   ┌──────▼──────┐
     │  35+ Metrics │                  │  7 Probes   │
     │  ─────────── │                  │  ─────────  │
     │  Fairness    │                  │  Hardware   │
     │  Data Quality│                  │  Carbon     │
     │  Performance │                  │  BOM        │
     │  Privacy     │                  │  Artifact   │
     │  Robustness  │                  │  Trace      │
     └──────┬──────┘                   └──────┬──────┘
            │                                 │
            └───────────────┬─────────────────┘
                            │
                     ┌──────▼──────┐
                     │  Dashboard  │
                     │  ─────────  │
                     │  Identity   │
                     │  Policy     │
                     │  Verify     │
                     │  Report     │
                     └─────────────┘
```

## Installation

<Tabs>
  <TabItem label="Core">
    ```bash
    pip install venturalitica
    ```
  </TabItem>
  <TabItem label="Fairness">
    ```bash
    pip install "venturalitica[metrics]"
    ```
    Adds [Fairlearn](https://fairlearn.org/) for advanced fairness metrics.
  </TabItem>
  <TabItem label="Dashboard">
    ```bash
    pip install "venturalitica[dashboard]"
    ```
    Adds [Streamlit](https://streamlit.io/) for the Glass Box dashboard.
  </TabItem>
  <TabItem label="Agentic">
    ```bash
    pip install "venturalitica[agentic]"
    ```
    Adds LangGraph, LangChain, and llama-cpp-python for agent workflows.
  </TabItem>
  <TabItem label="Green">
    ```bash
    pip install "venturalitica[green]"
    ```
    Adds [CodeCarbon](https://codecarbon.io/) for energy consumption tracking.
  </TabItem>
  <TabItem label="Full">
    ```bash
    pip install "venturalitica[full]"
    ```
    Installs all optional packages (metrics + dashboard + agentic + green + torch).
  </TabItem>
</Tabs>

Requires Python 3.11+.
