**AI Assurance a través de Cumplimiento como Código.**

Venturalítica transforma su código Python en **Evidencia Legal**. Mapea automáticamente sus métricas técnicas, auditorías de datos y registros de ejecución con la **Ley de IA de la UE (Artículos 9-15)** sin salir de su entorno local.

---

## Inicio rápido en 60 segundos

```bash
pip install venturalitica
```

Detecte sesgos en sus conjuntos de datos o modelos con una línea de código:

```python
import venturalitica as vl

# Ejecute una auditoría completa de AI Assurance en el escenario de préstamo integrado
results = vl.quickstart('loan')
```

Luego inicie el **Dashboard de AI Assurance** para explorar los resultados:

```bash
venturalitica ui
```

---

## Características clave

| Característica | Descripción |
| :--- | :--- |
| **`enforce()`** | Audite conjuntos de datos y modelos contra políticas OSCAL con más de 35 métricas integradas. |
| **`monitor()`** | Envuelva las ejecuciones de entrenamiento con recolección automática de evidencias (BOM, hardware, carbono). |
| **Dashboard de AI Assurance** | Flujo de trabajo regulatorio de 4 fases: Identidad, Política, Verificar, Informar. |
| **Política como Código** | Defina reglas de garantía en formato `assessment-plan` de OSCAL. |
| **Column Binding** | Desvincule las políticas de los esquemas mediante resolución de columnas basada en sinónimos. |
| **Soberanía Local** | Cero dependencia de la nube. Toda la ejecución se realiza localmente. |
| **Anexo IV** | Borrador automático de documentación técnica a partir de trazas locales. |
