**AI Assurance através de Compliance-as-Code.**

A Venturalítica transforma o seu código Python em **Evidência Legal**. Mapeia automaticamente as suas métricas técnicas, auditorias de dados e registos de execução para o **IA Act da UE (Artigos 9-15)** sem sair do seu ambiente local.

---

## Início rápido em 60 segundos

```bash
pip install venturalitica
```

Detete enviesamentos nos seus conjuntos de dados ou modelos com uma única linha de código:

```python
import venturalitica as vl

# Execute uma auditoria completa de AI Assurance no cenário de empréstimo integrado
results = vl.quickstart('loan')
```

Em seguida, inicie o **Dashboard de AI Assurance** para explorar os resultados:

```bash
venturalitica ui
```

---

## Principais Funcionalidades

| Funcionalidade | Descrição |
| :--- | :--- |
| **`enforce()`** | Audite conjuntos de dados e modelos contra políticas OSCAL com mais de 35 métricas integradas. |
| **`monitor()`** | Envolva as execuções de treino com recolha automática de evidências (BOM, hardware, carbono). |
| **Dashboard de AI Assurance** | Fluxo de trabalho regulatório de 4 fases: Identidade, Política, Verificar, Relatar. |
| **Handshake Criptográfico** | Verificação de integridade entre a política definida e a evidência técnica gerada. |
| **Política como Código** | Defina regras de garantia no formato `assessment-plan` do OSCAL. |
| **Column Binding** | Desvincule as políticas dos esquemas através de resolução de colunas baseada em sinónimos. |
| **Soberania Local** | Zero dependência da nuvem. Toda a execução é feita localmente. |
| **Anexo IV** | Rascunho automático de documentação técnica a partir de traços locais. |
